# Reinstall The Focuser

Re-mount the focuser onto the Etalon module as shown in the image. To ensure the focuser knobs are in a practical position, align them parallel with the Etalon’s pressure tuning knob. Ensure all three thumb screws securing the focuser are tightened before proceeding.
